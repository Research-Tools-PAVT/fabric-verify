The test-net folder contains the setup docker files and the chaincode folder
has the golang chaincode implementaion in cross-payment folder, payment.go file.

To run Tests : 

Open Terminal in chaincode/cross-payment folder 

	GO11MODULE= on go mod vendor
	make test

Run : 

Open a terminal in test-net folder and run 

	chmod 755 *.sh && yes | ./start.sh 

For details read the README.md file. 

GO MODULES are now supported. To install 
	
	GO11MODULE=on go mod vendor	
	
Add new commands in dump.json file and run assemble.sh to generate executable chaincode invokes.
